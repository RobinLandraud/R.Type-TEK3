Subject

For this projetct we have to make our own version or R-Type. But it's not only that...
We have to make a network game where one to four players will be able to play together.

Information

  • We used the SFML librairy for this project.
  • We did a whole game engine (ECS) permitting to add multiple graphical libraries thanks to encapsulation.
  • We add a scripting part, with ChaiScript for developper.
