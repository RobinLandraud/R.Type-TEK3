Detailed Description

Maps strings to Objects.

ChaiScript has a built in shortcut for generating Map objects:

Example:

eval> var m = ["a":1, "b":2];
[<a,1>, <b,2>]
eval> m.count("a");
1
eval> m.count("c");
0
eval> m.size();
2
