\subsubsection*{template$<$typename T$>$\newline
class chaiscript\+::detail\+::threading\+::\+Thread\+\_\+\+Storage$<$ T $>$}

Typesafe thread specific storage. If threading is enabled, this class uses a mutex protected map. If threading is not enabled, the class always returns the same data, regardless of which thread it is called from.
