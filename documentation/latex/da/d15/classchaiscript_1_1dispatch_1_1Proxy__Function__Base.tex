\doxysubsection{Detailed Description}
Pure virtual base class for all Proxy\+\_\+\+Function implementations Proxy\+\_\+\+Functions are a type erasure of type safe C++ function calls. At runtime parameter types are expected to be tested against passed in types. Dispatch\+\_\+\+Engine only knows how to work with Proxy\+\_\+\+Function, no other function classes.
