\doxysubsection{Detailed Description}
This file contains code necessary for thread support in Chai\+Script. If the compiler definition CHAISCRIPT\+\_\+\+NO\+\_\+\+THREADS is defined then thread safety is disabled in Chai\+Script. This has the result that some code is faster, because mutex locks are not required. It also has the side effect that the \mbox{\hyperlink{classchaiscript_1_1ChaiScript}{chaiscript\+::\+Chai\+Script}} object may not be accessed from more than one thread simultaneously.
